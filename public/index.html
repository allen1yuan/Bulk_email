Gmail Bulk Sender

Send an email to a list of recipients, one at a time, from your own Gmail account.

Login

Generate one at Google Account → Security → 2-Step Verification → App Passwords. Requires 2-Step Verification to be enabled.

Saved to a local file on this server (not the browser), never sent anywhere but Gmail. Anyone with access to this app can see it — only enable on a server you trust.

Compose

Use {handle} anywhere in the subject to auto-insert each recipient's Instagram handle.

Signature
Banner & Footer

If set, the banner image becomes a clickable link. Leave blank for a plain image or no banner.

Sending
to

A random delay in this range is picked before each send. Varying the gap looks less automated to Gmail's spam filters than a fixed interval — for small daily outreach batches, 5–15s is safer than the default. Min 0.5s, max 300s.

Note: Gmail limits regular accounts to roughly 500 sends per day.
